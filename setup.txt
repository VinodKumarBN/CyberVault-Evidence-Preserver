
🛡️ CYBERVAULT PRO - SYSTEM ENVIRONMENT SETUP CHECKLIST

[STEP 1] INSTALL Python (Version 3.10 or higher recommended)

[STEP 2] OPEN VS CODE TERMINAL AND RUN ENVIRONMENT INITIALIZATION:
    python -m venv venv

[STEP 3] ACTIVATE THE WORKSPACE ENVIRONMENT:
    Windows (Command Prompt):  .\venv\Scripts\activate
    Windows (PowerShell):      .\venv\Scripts\Activate.ps1
    macOS / Linux terminal:    source venv/bin/activate

[STEP 4] INSTALL DEPENDENCIES FROM THE REQUIREMENTS FILE:
    pip install -r requirements.txt

[STEP 5] LAUNCH INTERFACE FOR LOCAL TESTING AND DEMO:
    streamlit run app.py
